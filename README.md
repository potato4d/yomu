# Yomu

> The README explorer for node_modules.

[![Image from Gyazo](https://i.gyazo.com/d7772b6d638a2eee7c45cb4b410aa573.gif)](https://gyazo.com/d7772b6d638a2eee7c45cb4b410aa573)

## Installation

```bash
$ npm i -g yomu
```

## Usage

```bash
$ yomu <package-name>
```

### with less

```bash
$ yomu <package-name> | less -R
```
